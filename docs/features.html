Ban :

Kick :

j!kick [@user/username] {reason}

Functional Commands

Logging :
Sets up logging for your guild.
j!logging setchannel [name] // j!logging set [true/false]

Interactive Message :
Adjusts the dynamic messages.
j!im toggle [ban/kick/join/leave]
j!im setchannel [leave/join] [channel name]
j!im [join/leave/ban/kick] [message]
Variables : &guild&, &user&, &reason& (punishments)

AutoRole :
Joining users get automatically get assigned a role.
j!autorole [on/off]
j!autorole set [role name]

Prefix :
Adjust your guild's prefix.
j!prefix set [new prefix]

Permissions :
Your guild's permissions.
For more information, go here.

Purge :
Remove a defined amount of messages from a channel.
To make the message silent, add -s to the end of the command.
j!purge [amount of messages]
