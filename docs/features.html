Features


The features for Chad.


! Arguments surrounded by [] are required, {} is not.

! Make sure to replace j! with your guild's prefix.

! For extra help with each command, add help at the end of it.


Punishments

Ban :
Bans the selected user.
j!ban [@user/username] {reason}

Kick :
Kicks the selected user.
j!kick [@user/username] {reason}

Fun Commands

Cat Fact :
Gets a random cat fact.
j!catfact

Cat Gallery :
Gets a random cat picture.
j!catgallery

Eight Ball :
Responds to your questions.
j!8ball [question]

PhotoEditor :
Edits your photos, for you.
j!pe [blur] (include photo in message)

Russian Roulette :
Russian Roulette another user.
j!rrl [@user/username]

Random :
Gets a random quote, or a random number.
j!random [number/quote] {max value for number}

Functional Commands

Logging :
Sets up logging for your guild.
j!logging setchannel [name] // j!logging set [true/false]

Interactive Message :
Adjusts the dynamic messages.
j!im toggle [ban/kick/join/leave]
j!im setchannel [leave/join] [channel name]
j!im [join/leave/ban/kick] [message]
Variables : &guild&, &user&, &reason& (punishments)

Informational Commands

Guild Info :
Gets information about the guild.
j!guilfinfo

Help :
Gets all Chad's commands.
j!help

Chad :
Information about Chad.
j!chad

Reddit Top :
Gets all Chad's commands.
j!rtop [subreddit]

Reddit New :
Gets all Chad's commands.
j!rnew [subreddit]

Steam :
Gets statistics from Steam.
j!steam profile [steam name]
j!steam csgo [steam name] {lastmatch/kills/maps}

User Info :
Gets information about a user.
j!userinfo [@user/username]
